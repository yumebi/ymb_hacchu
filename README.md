# YMB 発注くん

発注書をかんたんに作成・印刷(PDF出力)できるローカルデスクトップアプリ。
Electron製、サーバー不使用、データは自分のPCにJSONファイルとして保存。

## 主な機能

- **発注書フォーム入力**: 発注元(自社)・宛先(取引先)・品目明細を入力するだけで
  和暦日付表示・小計/消費税(10%)/合計を自動計算した発注書レイアウトを表示。
- **発注元/宛先の複数登録**: 自社情報・取引先情報をそれぞれ複数登録してドロップダウンで
  切り替え可能。
- **明細行の自由編集**: 品名・数量・単位・単価を入力すると金額を自動計算。行の追加/削除が自由。
- **発注履歴の保存・一覧・再利用**: 保存した発注書は一覧に表示され、クリックで呼び出して
  編集・再印刷が可能。削除も可能。
- **PDF出力**: 「保存してPDF出力」で発注書を履歴に保存しつつ、その場でPDFファイルとして
  書き出し(印刷ダイアログ不使用、直接ファイル保存)。
- **本体の更新確認**: GitHub上の`version.json`と現在バージョンを比較し、起動時に
  自動チェック。新しいバージョンがあれば画面上部にダウンロード導線を表示。
- **データ保存**: 発注元/取引先/発注履歴はすべてローカルのJSONファイルに保存(サーバー通信なし)。

## 使い方

- 左パネル: 発注元/宛先の登録・選択、明細行の追加。
- 「新規発注書」: フォームをクリアして新しい発注書の入力を開始。
- 「履歴に保存」: 現在の内容を履歴に保存(No.が既存と同じ場合は上書き)。
- 「保存してPDF出力」: 履歴に保存した上でPDFファイルの保存先を選んで書き出し。
- 左パネル下部の「発注履歴」: 過去の発注書を「開く」で呼び出し、「削除」で除去。

## 開発環境のセットアップ

```
npm install
npm start
```

## 技術スタック

- **フレームワーク**: Electron(Node.js + Chromium)
- **フロントエンド**: `renderer/`配下の静的HTML/CSS/バニラJS(フレームワーク不使用)。
  IPCは`contextBridge`経由の`window.api`ホストオブジェクト呼び出し。
- **PDF出力**: `webContents.printToPDF`によるメインプロセス側での直接PDF生成。
- **データ永続化**: JSON(`data.json`)、`%APPDATA%\YMB 発注くん\`配下に保存(DB不使用)。
- **インストーラー**: electron-builder(NSIS、日本語UI)。ポータブル版(インストール不要の単体exe)も同梱。
- **CI/CD**: GitHub Actions(`.github/workflows/release.yml`)、masterへのpushで
  バージョン自動採番→ビルド→GitHub Release公開まで自動化。

データ保存先: `%APPDATA%\YMB 発注くん\data.json`

## ダウンロード

ビルド不要で使う場合は [Releases](https://github.com/yumebi/ymb_hacchu/releases/latest) から
以下のいずれかをダウンロードして実行する。

- `YmbHacchuKun-Setup-<version>.exe` — インストーラー版(日本語UI、インストール先選択可)
- `YmbHacchuKun-Portable-<version>.exe` — ポータブル版(インストール不要、単体exeを実行するだけ)

> **注意**: このexeはコード署名されていません。ダウンロード・実行時にWindows SmartScreenが
> 「不明な発行元」として警告を表示する場合があります。「詳細情報」→「実行」で続行できます。
